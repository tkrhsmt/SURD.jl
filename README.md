SURD.jl is a Julia package for exact computation with SURD (Synergistic-Unique-Redundant Decomposition).

## Installation

You can install SURD.jl using Julia's package manager. Open the Julia REPL and run the following commands:

```julia
using Pkg
Pkg.add("https://github.com/tkrhsmt/SURD.jl")
```

## Usage

The usage instructions and examples can be found in the [documentation](https://tkrhsmt.github.io/SURD.jl).
